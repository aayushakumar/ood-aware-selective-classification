The Challenge

Why Current Methods Fail

Toxicity classifiers face critical challenges when deployed across different platforms and communities

‚ö†Ô∏è

OOD Score Inversion

When deploying from low-toxicity (Wikipedia) to high-toxicity (Twitter/Gab) domains, OOD energy scores become inverted ‚Äî making abstention decisions completely wrong.

Fairness Disparities

Models have significantly higher false positive rates on comments mentioning identity groups. Naive abstention makes this worse by burden-shifting to human review.

FPR on identity groups +27%

No Target Labels

At deployment time, we don't have target toxicity labels. We need a way to assess OOD reliability using only domain membership information.

Pipeline Overview

1

Compute OOD Detectability Metric (ODM)

Measure separability between source and target OOD scores using AUROC. Detect direction (normal vs inverted) from mean comparison. No target labels needed.

2

Adaptive Policy Selection

If separability > œÑ: use both confidence + OOD (with direction correction). If separability ‚âà 0.5: fall back to confidence-only policy.

3

Fairness-Constrained Threshold Optimization

Grid search for thresholds that maximize accuracy while enforcing 4 constraints: FPR gap, abstention gap, coverage, min group coverage.

4

Deploy with Anti-Leakage Protocol

Use source statistics for OOD standardization. Never compute stats from test data. Evaluate on held-out target-test splits.

Key Insights

What We Discovered

Novel insights about OOD detection in cross-domain toxicity classification

üîÑ

OOD Inversion is Systematic

When source has low toxicity and target has high toxicity, energy scores invert predictably. This is not random ‚Äî it correlates with toxicity rate difference.

Source ‚Üí Target Direction Sep.
Jigsaw ‚Üí HateXplain ‚¨áÔ∏è Inverted 0.936
Civil ‚Üí HateXplain ‚¨áÔ∏è Inverted 0.974
HateXplain ‚Üí Jigsaw ‚¨ÜÔ∏è Normal 0.934
‚úÖ

High Separability + Inverted = Useful!

The key insight: high separability with inverted direction is still useful ‚Äî just flip the scores! Only low separability (‚âà0.5) is truly useless regardless of direction.

Accuracy gain from inversion correction +8.3%
‚ö†Ô∏è

Naive OOD Causes Burden-Shifting

Without inversion correction, Naive OOD abstains on the wrong samples ‚Äî abstaining 65% on identity groups vs 50% on others. This unfairly routes minorities to expensive human review.

Naive OOD abstention gap 15%
‚öñÔ∏è

Both Gaps Must Be Constrained

Constraining only FPR gap is insufficient for fairness. We must also control the abstention gap to prevent shifting the burden of uncertainty to specific identity groups.

Our abstention gap 5%
